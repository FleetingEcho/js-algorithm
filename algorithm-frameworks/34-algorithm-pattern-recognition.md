## 📊 复杂度速查

```
O(1):   数组随机访问、哈希表查增删
O(log n): 二分搜索、平衡树
O(n):   线性遍历、前缀和
O(n log n): 排序、基于比较的算法
O(n²):  双层循环、朴素 DP
O(2ⁿ):  回溯、子集枚举（n ≤ 20）
O(n!):  全排列（n ≤ 10）
```

## 💡 面试通用策略

```
面对一道题，按这个顺序思考：
① 暴力解是什么？（别跳步，先想最简单的）
② 有没有重复计算？（有 → DP/备忘录）
③ 数据有什么性质？（有序？单调？有环？）
④ 需要最优解还是所有解？（最优→DP/BFS/贪心，所有→回溯/DFS）
⑤ 时间空间复杂度要求？（N≤20→回溯，N≤10⁵→O(n)/O(n log n)）
```
