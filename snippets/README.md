## 各文件章节

所有三个文件均按相同章节顺序组织：

```
1. 数组基础         — 初始化、交换、切片、转置、推导
2. 双指针           — 左右夹逼、快慢、三指针
3. 滑动窗口         — 可变窗口、定长窗口
4. DFS 模板         — 图、网格、回溯
5. BFS 模板         — 单源、多源、按层
6. 排序技巧         — 自定义排序、计数排序
7. 字符串           — 字符操作、拆分、查找
8. 数据结构         — 堆、哈希、双端队列、有序集
9. 数学工具         — 快速幂、GCD、质数筛
10. 语言特有技巧    — 各语言特色写法和常见陷阱
```
